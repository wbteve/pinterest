CKFinder Interface Overview

The CKFinder interface has been designed to be clear, familiar to our end users, and easy to learn and use. Most features can be handled with a mouse click and context menus.

The following is a screenshot of CKFinder:

  1. Folders Pane: contains the "tree view" of the folders where you can navigate. Folders are ways to organize files better.
  2. Files Pane: lists the files available in the selected folder.
  3. Toolbar: a series of buttons that can be clicked to quickly execute specific functions.
  4. Status Bar: space used to display information regarding the select file, the total number of files in the folder, etc.
  5. Context Menu: a series of buttons that can be used to execute specific tasks in the object that has been clicked. The available options dynamically change, depending on which kind of object is clicked.
